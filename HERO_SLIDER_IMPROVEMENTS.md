# Hero Slider 轮播图优化记录

## 第二次优化（2026年6月22日）

### 修复的问题
**图片切换留白问题**：移除 `AnimatePresence` 的 `mode="wait"` 参数，实现图片交叉淡入淡出，避免切换时的空白间隙。

### 优化内容

#### 1. 修复切换留白
- **问题**：使用 `mode="wait"` 导致旧图片完全退出后才显示新图片，中间有明显留白
- **解决**：移除 `mode` 参数，使用默认的交叉淡入淡出效果
- **效果**：新旧图片平滑过渡，无留白，观感更佳

#### 2. 按照官网还原字体布局
- **官网结构分析**：
  ```html
  <h1>MINAMI UMEZAWA<span>OFFICIAL WEBSITE</span></h1>
  ```
  - 主标题和副标题在同一个 h1 元素内
  - 副标题使用 span 标签
  
- **我们的实现**：
  ```tsx
  <motion.h1 className="text-white">
    <span className="block text-4xl md:text-6xl lg:text-7xl font-thin tracking-[0.35em] mb-4">
      MINAMI UMEZAWA
    </span>
    <span className="block text-xs md:text-sm tracking-[0.5em] font-light opacity-80">
      OFFICIAL WEBSITE
    </span>
  </motion.h1>
  ```

#### 3. 字体布局细节调整
- **主标题**：
  - 字间距调整为 `tracking-[0.35em]`（从 0.4em 微调）
  - 保持字重 200
  - 底部间距 `mb-4`
  
- **副标题**：
  - 字号缩小为 `text-xs md:text-sm`（更接近官网）
  - 字间距调整为 `tracking-[0.5em]`（从 0.6em 调整）
  - 透明度 80%
  - 字重 300

#### 4. 结构优化
- 将主副标题从分离的两个元素合并到一个 h1 内
- 使用 span 标签分别包裹主标题和副标题
- 移除了副标题的独立 motion 动画（统一在 h1 上）
- 简化了动画逻辑

## 第一次优化（2026年6月22日）

### 优化内容

### 1. 轮播切换效果改进
- **原实现**：左右滑动切换 (x 轴移动)
- **新实现**：淡入淡出 + 轻微缩放效果
- **改进点**：
  - 移除了 x 轴滑动，改为纯透明度渐变
  - 添加微妙的缩放效果 (1.05 → 1.0)，营造 Ken Burns 效果
  - 透明度过渡时长：1.2秒
  - 缩放动画时长：8秒（缓慢放大，更有电影感）

### 2. 动画时长优化
- **原设置**：4000ms (4秒)
- **新设置**：5000ms (5秒)
- **原因**：给用户更多时间欣赏每张图片，更符合日系艺人官网的优雅节奏

### 3. 字体布局优化
#### 主标题 (MINAMI UMEZAWA)
- 字体大小调整：`text-4xl md:text-6xl lg:text-7xl`（比原来略小）
- 字重优化：`font-thin` + `fontWeight: 200`（更轻盈）
- 字间距增加：`tracking-[0.4em]`（从 0.3em 增加到 0.4em）
- 底部间距增加：`mb-6`（从 mb-4 增加）
- 动画优化：
  - 初始 Y 偏移减少到 20px（更微妙）
  - 延迟 0.3秒（更从容）
  - 时长 1秒（更平滑）
  - 每次切换都重新触发动画（添加 key）

#### 副标题 (OFFICIAL WEBSITE)
- 字体大小调整：`text-sm md:text-base lg:text-lg`（更精致）
- 透明度调整：`text-white/80`（从 70% 提高到 80%）
- 字间距增加：`tracking-[0.6em]`（从 0.5em 增加）
- 字重设置：`font-light` + `fontWeight: 300`
- 动画延迟：0.5秒（比主标题晚 0.2秒）
- 每次切换都重新触发动画（添加 key）

### 4. 界面简化
- **移除元素**：
  - 左右箭头导航按钮
  - 暂停/播放按钮
- **保留元素**：
  - 圆点指示器（优化样式）
  - 滚动提示（优化动画）
- **原因**：日系艺人官网通常采用极简设计，减少干扰元素

### 5. 圆点指示器优化
- 间距增加：`space-x-3`（从 space-x-2 增加）
- 当前状态：`w-8 h-1.5`（横向拉长的矩形，更现代）
- 非当前状态：`w-1.5 h-1.5`（小圆点）
- 过渡时长：500ms（从 300ms 增加，更平滑）
- 可点击切换，无方向判断逻辑

### 6. 滚动提示优化
- 出现延迟：1.5秒（从 1.2秒增加）
- 淡入时长：1秒（更柔和）
- 动画幅度：10px（从 8px 增加）
- 动画时长：2.5秒（从 2秒增加，更缓慢）
- 文字透明度降低：`text-white/30`（更低调）
- 线条高度增加：`h-16`（从 h-12 增加）
- 线条透明度调整：`from-white/40`

## 技术细节

### 动画变体 (Variants)
```typescript
const variants = {
  enter: {
    opacity: 0,
    scale: 1.05,  // 进入时轻微放大
  },
  center: {
    opacity: 1,
    scale: 1,     // 显示时正常大小
  },
  exit: {
    opacity: 0,
    scale: 1,     // 退出时保持大小，只淡出
  },
};
```

### 过渡配置 (Transition)
```typescript
transition={{
  opacity: { duration: 1.2, ease: "easeInOut" },  // 淡入淡出
  scale: { duration: 8, ease: "linear" },          // 缓慢放大
}}
```

## 预期效果
1. **视觉感受**：更优雅、更平滑、更有电影感
2. **用户体验**：减少视觉干扰，专注于内容本身
3. **品牌调性**：符合日系艺人官网的简约高雅风格
4. **性能表现**：淡入淡出比滑动更流畅，无需处理方向状态

## 兼容性
- 响应式设计保持不变
- 桌面端和移动端图片分别加载
- 主题切换功能不受影响
- 无障碍支持（aria-label）保持完整
